Installable PWA

CGN NOW

Install CGN NOW for direct app-style access to CGN News, sports, weather, radar and account features.

Continue in browser

When installed, the app name appears as CGN NOW and launches to the News page.

iPhone / iPad

  1. Open this page in Safari.
  2. Tap the Share icon.
  3. Select Add to Home Screen.
  4. Tap Add, then launch CGN NOW from your Home Screen.

Android / Desktop

  1. Use the browser install button when it appears.
  2. Or choose Install app / Add to Home screen from the browser menu.
  3. Open CGN NOW as a standalone app.
